BudgetPlanner Pro
A powerful desktop financial planner for structured management of income, expenses, and savings goals.
The project was developed to combine realistic budget planning, clean Python architecture, and user-friendly desktop UIs.

Tech Stack
Python 3.10+
Tkinter – GUI
matplotlib – Charts
JSON – Persistent data storage
CSV / PDF – Data export
reportlab – PDF creation

Installation & Start
Requirements
Python 3.10 or higher

Starting the application
git clone https://github.com/DEIN_USERNAME/budget-planner.git
cd budget-planner
python app.py

This project was developed to:
realistically map real financial planning
combine complex logic with a desktop UI
implement clean data persistence without an external database
write Python code in a structured, maintainable, and extensible way
I started this project to help friends and family in their everyday lives and give them a useful tool for managing their finances.
